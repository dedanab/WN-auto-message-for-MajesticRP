Инструкция по установке:
1. Ставьте сначала python 3.9.6 + patch
2. Запускайте install.bat
3. Редактируете settings.cfg и заменяйте время и текст
4. Далее включаете английский язык
5. Далее запускаете gtanewsbot.py

Бот отправляет в определенное время определенный текст и далее делает скриншот. Подойдет для фарма зеленок в WN. Кто играет на маджестике поймет
